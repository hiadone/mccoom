카트리지

6종 카트리지로 피부층별 섬세한 에너지 전달,
다양한 깊이에 응고존을 형성시킵니다.

두 가지 타입의 초음파 카트리지

도트 타입 카트리지 간격이 1~2mm까지 조절 가능하다면,
리니어 카트리지는 도트 사이의 간격이 0mm로
빈틈이 적고 고르게 에너지를 전달합니다.
  • Dot (점) 타입
  • Linear (선) 타입
새로워진 선타입 리니어펌 리프팅을 만나보세요.

작은 부위에도 시술가능

좁은 부위에도 시술 가능한 QS 2.0카트리지는
폭의 길이가 8.6mm으로서 눈썹 등 작은 부분에도 시술이 가능합니다.
  • width : 12.5mm
  • width : 8.6mm

시술원리

집속형초음파 트랜스듀서에서 출력된 초음파 에너지의
열작용을 이용하여
조직을 응고시켜 EYEBROW 리프팅을 합니다.
피부 얕은 층부터 깊은 층까지 다양한 깊이에 응고존 형성이 가능합니다.
